# Weekly Review Workflow

Execute these steps IN ORDER. Complete each step fully and show the required OUTPUT before moving to the next step.

**Best run on Friday afternoon.**

---

## Step 1: Load Context

Read these files:
- `second-brain/profile.md` (objectives)
- `second-brain/company.md` (company goals)
- `second-brain/projects.md` (all active projects)
- `outbox/SOD/` (this week's daily briefings)
- `logs/` (this week's logs)
- `second-brain/decisions.md` (recent decisions)
